# Travel App UI In Flutter For Android & iOS
Travel App UI Template made in Flutter for both Android & iOS.

# Running The Project: 
<a href="https://flutter.dev">Install Flutter</a>

# Commands: <br/>
flutter pub get <br/>
flutter run

## Packages Used
1)google_fonts

2)carousel_slider

3)flutter_swiper
